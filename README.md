# 📚 Student Study Planner

A full-stack web application for tracking university units and assignments.
Built as part of my Bachelor of Information Technology at Deakin University,
applying skills from Database Fundamentals (SIT103), Responsive Web Apps 
(SIT120), and Introduction to Programming (SIT102).

## ✨ Features
- Add and manage university units with trimester and status tracking
- Add assignments with due dates, weight percentages, and status
- Dashboard showing total units, completed units, and assignment stats
- Update unit and assignment status with instant dropdown changes
- Delete units and assignments

## 🛠️ Technologies Used
- Python 3
- Flask (web framework)
- Flask-SQLAlchemy (database ORM)
- SQLite (database)
- HTML & CSS (frontend)

## 🚀 How to Run
1. Install dependencies:
   pip install flask flask-sqlalchemy
2. Run the app:
   py planner.py
3. Open your browser at https://student-study-planner-phmj.onrender.com
